How to update a policy

  1. Make edits in the working copy stored offline.
  2. Send the revised document to Michelle for approval.
  3. Once approved, replace the file in the repo and update this page if the filename changes.

Note: keep all safeguarding and sensitive personnel documents restricted in the staff area or Protected Resources. If you need the file visible to parents or partners, create a public summary version without protected details.
