Protected resources and staff access

Certain internal documents, staff rotas, sensitive case records and other protected resources are not published on the public website. This is to protect confidentiality and comply with data protection and safeguarding requirements.

What is protected

  • Client records and case notes
  • Staff-only rotas and internal timesheets
  • Protected policy versions that include contact details and secure forms
  • Passwords and internal credentials

How staff request access

To request access to any protected resource, please email one of the leads below. We will verify requests and provide secure access links or file transfers out of band (for example via a secure drive link or a private portal).

Request access:
Michelle — Michelle.Pascoe@meadowpathwayscornwall.com
Zoe — Zoe.Waitz@meadowpathwayscornwall.com
If you are an authorised staff member, include your staff email and the resource(s) you need in the email.

Secure sharing practice

We will not send protected documents by unencrypted email. Approved methods include: secure file sharing (private drive link), password-protected archived file sent by secure channel, or secure internal portal link provided after verification.

If you believe something that should be protected is published publicly, please contact us immediately so we can remove it and arrange secure sharing if appropriate.
